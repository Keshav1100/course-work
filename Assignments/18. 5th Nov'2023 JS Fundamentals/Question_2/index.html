Q. Describe the usage of the comma operator in JavaScript and provide an example.

Ans.

The comma operator in JavaScript allows multiple expressions to be combined into a single expression. It evaluates each of its operands from left to right and returns the value of the last operand. While it's not commonly used, it can be helpful in certain situations where multiple expressions need to be executed sequentially.
