Q.

Explain the role of the following key properties in the Flexbox layout model:
  1. justify-content
  2. align-items
  3. gap
  4. flex-direction
  5. flex-wrap

A.

  1. justify-content:

    Role: This property controls the alignment of items along the main axis (the axis defined by the flex-direction property).

  2. align-items:

    Role: This property controls the alignment of items along the cross axis (the axis perpendicular to the main axis defined by flex-direction).

  3. gap:

    Role: This property controls the alignment of items along the cross axis (the axis perpendicular to the main axis defined by flex-direction).

  4. flex-direction:

    Role: This property defines the direction in which the main axis of the flex container runs.

  5. flex-wrap:

    Role: This property defines whether flex items are forced onto a single line or can wrap onto multiple lines along the cross axis.
