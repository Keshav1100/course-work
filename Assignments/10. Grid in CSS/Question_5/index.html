Q. Explain the difference between justify-items and justify-self using code examples.

A.

In CSS Grid, justify-items and justify-self are properties that control the horizontal alignment of grid items within a grid container. Below is the breakdown of both with examples:

'justify-items' Property:

'justify-items' is used in the parent grid container to align all the grid items within it along the inline (row) axis.

'justify-self' Property:

'justify-self' is used within individual grid items to override the alignment set by 'justify-self' . It allows you to specifically control the horizontal alignment of each grid item.
