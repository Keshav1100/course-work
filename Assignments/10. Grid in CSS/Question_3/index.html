Q. Explain the use of grid-auto-row and grid-auto-column using code examples.

A.

'grid-auto-row' and 'grid-auto-column' are properties used in CSS Grid Layout to define the size and behavior of implicitly created rows and columns within a grid container. They come into play when you have more grid items than explicitly defined rows or columns.

Example:

1. 'grid-auto-row'

This property defines the size of implicitly created rows in a grid. For instance, if our grid has only three explicitly defined rows, and we place a fourth grid item, grid-auto-row will determine the size of this newly added row.

The grid-auto-rows property will set the size of any additional rows that are created beyond these three explicitly defined rows. So, when the fourth grid item is added to the grid, a new row will be created, and its height will be 80px as per the grid-auto-rows property.

2. 'grid-auto-column'

Similarly, grid-auto-column defines the size of implicitly created columns in a grid layout when the number of grid items exceeds the explicitly defined columns.
