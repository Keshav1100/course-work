Q. Explain the z-index, with a code example.

A.

The z-index property in CSS is used to control the stacking order of positioned elements along the z-axis (front-to-back). This property is particularly useful when we have overlapping elements on a webpage, and we want to specify which element should appear on top

Example:

Box-2
