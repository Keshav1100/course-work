Some common HTML tags

  1. <html> : This tag is the root element of an HTML document, enclosing all other elements. It defines the document as an HTML file.

  2. <head> : Within this tag, you include meta-information about the document, such as the title, character encoding, and linked stylesheets or scripts.

  3. <title> : This tag sets the title of the webpage, which appears in the browser's title bar or tab.

  4. <meta> : The <meta> tag provides metadata about the HTML document, including character encoding and authorship information.

  5. <link> : Used to link external resources like CSS stylesheets, making it crucial for styling and formatting web pages.

  6. <style> : This tag contains inline CSS code to define the document's styling within the HTML file.

  7. <script> : Allows you to embed or reference JavaScript code, enhancing interactivity and functionality.

  8. <body> : Encloses the visible content of the webpage, including text, images, links, and other media.

  9. <h1> to <h6> : These tags define headings of varying levels, with <h1> being the highest and <h6> the lowest. They help structure content hierarchically.

  10. <p> : Stands for "paragraph" and is used for defining text paragraphs within the document.

  11. <a> : The anchor tag creates hyperlinks, allowing users to navigate to other web pages or resources.

  12. <img> : Used to embed images in the document, requiring attributes like src for the image source and alt for alternative text.

  13. <ul> : Defines an unordered list, often used for creating bulleted lists of items.

  14. <ol> : Represents an ordered list, used for numbered lists to indicate item sequence.

  15. <li> : Nested within <ul> or <ol>, this tag defines list items within both unordered and ordered lists.

  16. <div> : A versatile container used for grouping and styling elements. It's often used in CSS for layout purposes.

  17. <span> : Similar to <div>, but used for inline styling and grouping of text or inline elements.

  18. <table> : Used for creating tables to organize data, with child tags like <tr> (table row), <th> (table header), and <td> (table data).

  19. <form> : Encloses input elements like text fields, buttons, and checkboxes, facilitating user interactions and data submission.

  20. <input> : This tag is used within <form> elements to create various types of input fields, such as text, radio buttons, and checkboxes.
