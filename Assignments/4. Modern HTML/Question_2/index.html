Q. What are HTML Entities? List out 5 commonly used HTML entities.

A. HTML entities are used to represent special characters in HTML. They are used because some characters are reserved in HTML, and cannot be used in the text of a document without being interpreted as markup. For example, the less than sign (<) is used to start an HTML tag, so if you want to display the less than sign in your text, you must use the HTML entity <.

HTML entities are created by placing an ampersand (&) at the beginning of the entity name, and a semicolon (;) at the end. For example, the HTML entity for the copyright symbol is ©.

Here is a list of 5 commonly used HTML entities:
  • &amp - ampersand
  • &lt; - less than
  • &gt; - greater than
  • &quot; - quotation mark
  • &copy; - copyright symbol

HTML entities are a powerful tool that can be used to display any character in your HTML document. They are especially useful for displaying special characters and characters that are not available on your keyboard.
