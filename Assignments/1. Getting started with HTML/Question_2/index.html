Question: Explain the purpose of comments in HTML and provide an example of how to use comments in an HTML document.

Answer:Comments in HTML serves as a way to include some explanatory notes within the code that are not visible to the end user when the web page is rendered in a web browser.
They also help in the following:

  1. They help in documentation of the code like the structure of a webpage or some notes regarding purpose of some elements.

  2. They help us in case we are debugging any issue and need to disable or delete some specific section of the page.

  3. It helps in explaining the code to different developers who are working on the same project.
